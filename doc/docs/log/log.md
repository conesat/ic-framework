# Ic Framework 更新日志

## 1.0.0  (2025-07-18)
- 🎉 正式开源
- 集成 Spring Boot 3.5.3+，最低支持 JDK 21
- 支持多端集成（中台、小程序、App）
- 内置代码生成器，提升开发效率
- 集成权限认证与多种缓存实现
- IcMybatis：mybatis 增强，简化 SQL 调整
- Dber：自动实体 DDL，支持 SQL 版本升级
- 集成角色权限体系
- 提供完整的文档与示例工程
- 开源社区支持（GitHub、Gitee）
